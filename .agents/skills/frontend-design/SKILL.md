# frontend-design

Create distinctive, production-grade frontend interfaces with high design quality.

## Use When
- Building web components/pages.
- Styling or improving existing UI.
- Designing dashboards, landing pages, and app screens.

## Design Expectations
- Avoid generic boilerplate layouts.
- Keep typography, spacing, and hierarchy intentional.
- Prioritize accessibility and responsive behavior.
- Use clear visual direction for color, motion, and component structure.

## Workflow
1. Understand product goal and target users.
2. Define visual direction and interaction model.
3. Implement reusable components and responsive layout.
4. Validate accessibility and edge states.
