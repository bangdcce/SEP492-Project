UC ID and Name: UC-02 View Public Landing Page
Created By: SonNT Date Created:
Primary Actor: Guest Secondary Actors:
Trigger: The guest enters the website domain URL or navigates to the root address.
Description: Allow the guest to view the main landing page containing general platform information, featured services, and navigation options (like Login/Register) without authentication.
Preconditions: PRE1: The internet connection is available on the guest's device.
PRE2: The web server is online and reachable.
Post-conditions: POS1: The landing page UI is fully rendered with correct information.
POS2: Navigation links (Register, Login, About Us) are active and clickable.
Normal Flow: A. View Page Successfully
  1. The guest enters the website URL in the browser.
  2. The system receives the request and retrieves public content (banners, text, public listings).
  3. The system loads the UI components (Header, Footer, Hero Section).
  4. The system displays the Landing Page to the guest.
  5. The guest scrolls or interacts with public information elements.
Alternative Flows: A5. Slow Network / Timeout
  1. The system takes too long to fetch assets.
  2. The system displays a skeleton loader or a "Loading..." state.
  3. Once loaded, resume at Normal Flow step 4.
