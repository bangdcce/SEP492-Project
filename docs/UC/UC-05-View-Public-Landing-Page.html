UC ID and Name: UC-05 View Public Landing Page
Created By: SonNT Date Created: 2026-01-22
Primary Actor: Guest Secondary Actors: Web Application
Trigger: The guest opens the platform website.
Description: Allow a guest to view general information about the platform and navigate to Sign In or Sign Up.
Preconditions: PRE1: Internet connection is available.
PRE2: The website is accessible.
Post-conditions: POS1: The landing page content is shown.
POS2: The guest can choose to sign in or create an account.
Normal Flow: A. View Public Landing Page
  1. The guest opens the website.
  2. The system displays public information such as platform overview and key features.
  3. The system displays navigation options like "Sign In" and "Sign Up".
  4. The guest reads the information and may choose a navigation option.
Alternative Flows: A2. Slow network
  1. The page loads slowly due to network conditions.
  2. The system shows a loading state until content appears.
Exceptions: E1: The website cannot be reached; the guest can retry later.
Priority: Medium
Frequency of Use: High
Business Rules: N/A
Other Information: Public content should not require the user to sign in.
Assumptions: N/A
