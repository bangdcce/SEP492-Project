UC ID and Name: UC-82 View List User Account
Created By: An Date Created: 25/01/2026
Primary Actor: Admin Secondary Actors: System
Trigger: Admin navigates to the "User Management" section.
Description: Allows the Admin to view a paginated list of all users registered in the system. This list provides an overview of the user base, including basic details like Name, Email, Role (Client/Broker/Freelancer), Account Status (Active/Banned), and Registration Date.
Preconditions: PRE1: The user is logged in as Admin.
Post-conditions: POS1: A table of users is displayed.
Normal Flow: A. View User List
  1. Admin clicks "Users" in the request sidebar.
  2. System queries the database for user records (with pagination).
  3. System displays the list with columns: User ID, Full Name, Email, Role, Status, Join Date.
  4. Admin navigates through pages (Next/Previous).
Alternative Flows: A2. Search User
  • At step 2, Admin enters a keyword (Name/Email) in the search bar.
  • System filters the list matching the keyword.
A2. Filter by Role
  • At step 2, Admin selects "Broker" from the Role dropdown.
  • System shows only Broker accounts.
Exceptions: EXC-01: No Data
  • Database empty (unlikely).
  • System shows "No users found".
Priority: High
Frequency of Use: High
Business Rules: BR-50: Only Admins can view the full user list with emails.
Other Information: Admins use this list as the entry point for Ban/Unban (UC-84) or View Detail (UC-83).
Assumptions: N/A
