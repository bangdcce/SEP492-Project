UC ID and Name: UC-39 View Subscription
Created By: An Date Created: 25/01/2026
Primary Actor: User (Client, Broker, Freelancer) Secondary Actors: System
Trigger: User accesses the "Membership" or "Billing" section of their profile settings.
Description: Allows the user to view the details of their current subscription plan. This includes the plan name (e.g., Free, Pro), validity period, next billing date, and active benefits (like lower fees or priority support).
Preconditions: PRE1: The user is logged in.
Post-conditions: POS1: Current subscription details are displayed.
Normal Flow: A. View Current Subscription
  1. User navigates to Settings > Billing.
  2. System retrieves current subscription data.
  3. System displays:
    • Active Plan: System highlights the current tier.
    • Status: Active / Overdue / Canceled.
    • Expiration/Renewal Date.
    • Payment Method linked.
Alternative Flows: A2. View History
  • At step 2, User clicks "Invoice History".
  • System lists past payments.
A2. Upgrade Plan
  • At step 2, User views benefits of higher tiers and clicks "Upgrade" (triggers UC-40).
Exceptions: EXC-01: Info Unavailable
  • Sync error with billing provider.
  • System displays cached info or "Service temporarily unavailable".
Priority: Medium
Frequency of Use: Low
Business Rules: BR-90: Default plan is "Free" if no active subscription exists.
Other Information: N/A
Assumptions: N/A
