UC ID and Name: UC-83 View Account Detail
Created By: An Date Created: 23/01/2026
Primary Actor: Admin Secondary Actors: System
Trigger: Admin selects a specific user from the User Management List.
Description: Allows the Admin to view comprehensive details of a specific user account. This includes personal info, KYC status/documents, Trust Score history, role distribution (Client/Broker/Freelancer), and current account status (Active/Banned/Restricted).
Preconditions: PRE1: Admin is logged in.
PRE2: The user account exists.
Post-conditions: POS1: All detailed information is displayed securely.
Normal Flow: A. View User Details
  1. Admin clicks on a specific row in the User List.
  2. System loads the User Profile from the database.
  3. System displays sections:
    • Basic Info (Name, Email, Phone, Created At)
    • KYC (ID Card Image, Status)
    • Roles & Permissions
    • Trust Score Metrics (Projects Completed, Disputes Lost)
    • Status Information (Active, Banned)
Alternative Flows: A3. View Related Activity
  • At step 3, Admin clicks "View Activity Log" from this page.
  • System redirects to System Log filtered by this User ID (UC-85).
Exceptions: EXC-01: User Not Found
  • Data integrity error.
  • System displays 404 error.
Priority: High
Frequency of Use: Medium
Business Rules: BR-50 (Admin Access Only), BR-39 (Privacy laws - Admin can view private data but must be logged).
Other Information: N/A
Assumptions: N/A
