UC ID and Name: UC-47 View Wallet
Created By: An Date Created: 25/01/2026
Primary Actor: Client, Broker, Freelancer Secondary Actors: System
Trigger: User clicks the Wallet icon/link in the navigation bar.
Description: Allows the user to view their internal financial wallet. This includes the Total Balance, Available Balance (withdrawable), and Locked Balance (in Escrow or pending withdrawal). It also shows a summary of recent transactions.
Preconditions: PRE1: The user is logged in.
Post-conditions: POS1: Wallet balances and transaction history are displayed.
Normal Flow: A. View Wallet Dashboard
  1. User clicks on "Wallet".
  2. System retrieves wallet entity associated with the User ID.
  3. System calculates balances:
    • Available: Liquid funds.
    • Locked: Funds held in active Milestones/Escrow.
  4. System displays the Balance Card.
  5. System lists recent transactions (Deposits, Payouts, Escrow Holds).
Alternative Flows: A5. Filter Transactions
  • At step 5, User filters by "Income" or "Expense".
  • System updates the list.
Exceptions: EXC-01: Wallet Suspended
  • User's financial privileges are frozen (suspected fraud).
  • System displays "Wallet Frozen" alert and hides actions (Withdraw/Deposit).
Priority: High
Frequency of Use: High (Daily)
Business Rules: BR-45: System creates a Wallet automatically upon User Registration.
Other Information: Display currency is typically USD.
Assumptions: N/A
