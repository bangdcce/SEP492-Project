UC ID and Name: UC-49 View List Cashout Item
Created By: An Date Created: 23/01/2026
Primary Actor: Freelancer, Broker Secondary Actors: System
Trigger: User accesses the "Wallet" or "Earnings" section.
Description: Allows the user to view a history of their cashout (withdrawal) requests. The list displays the amount, date requested, method, fee, and current status (e.g., Pending, Processing, Completed, Rejected).
Preconditions: PRE1: The user is logged in.
PRE2: The user has a wallet account.
Post-conditions: POS1: The list of cashout items is displayed sorted by date (newest first).
Normal Flow: A. View Cashout History
  1. User navigates to "Wallet".
  2. User selects the "Cashout History" tab.
  3. System retrieves payout requests from the database.
  4. System displays the list with columns: Request ID, Date, Amount, Status, Method.
Alternative Flows: A3. Empty History
  • At step 3, If user has never cashed out, display "No withdrawal history found."
A4. Filter by Status
  • At step 4, User filters by "Pending" to check active requests.
Exceptions: EXC-01: Service Error
  • System fails to load data.
  • Displays error message.
Priority: Medium
Frequency of Use: Low (Weekly/Monthly)
