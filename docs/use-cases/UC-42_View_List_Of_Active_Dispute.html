UC ID and Name: UC-42 View List Of Active Dispute
Created By: An Date Created: 23/01/2026
Primary Actor: Client, Freelancer, Broker Secondary Actors: System
Trigger: User navigates to the "My Disputes" or "Resolution Center" section.
Description: Allows the user to view a list of ongoing disputes they are involved in, either as the person who raised the dispute or the defendant. This serves as the central hub for tracking the progress of resolution, viewing status updates, and accessing detailed dispute information.
Preconditions: PRE1: The user is logged into the system.
PRE2: The user is involved in at least one active or past dispute.
Post-conditions: POS1: A list of the user's disputes is displayed.
POS2: The current status (e.g., Pending Evidence, Under Review) is clearly visible.
Normal Flow: A. View Dispute List
  1. User accesses the Resolution Center from the main menu.
  2. System retrieves all disputes linked to the User ID (either as Plaintiff or Defendant).
  3. System displays the list of disputes.
  4. For each item, the system shows: Related Project, Dispute Reason, Opposing Party Name, Amount in Dispute, and Current Status.
  5. User selects a specific dispute to view more details (extensions to UC-44).
Alternative Flows: A2. No Disputes
  • At step 2, if the user has no dispute history, the system displays an empty state message "No active disputes found".
A3. Filter by Status
  • At step 3, User clicks on "Resolved" tab to view closed cases.
  • System filters the list to show only resolved disputes history.
Exceptions: EXC-01: System Error
  • Failed to fetch data.
  • System displays "Temporary error loading disputes".
Priority: High
Frequency of Use: Low (Only when issues arise)
Business Rules: Users can strictly only view disputes they are a party to (No IDOR).
Other Information: This use case extends to "Create Dispute" (UC-43) and "View Dispute Information" (UC-44).
Assumptions: N/A
